# RentEase

RentEase is a full-stack property rental platform that allows users to list, browse, and book rental properties online. The platform provides an easy way for property owners to showcase listings and for users to find suitable rental homes.

## Live Demo

https://rentease-kl2m.onrender.com

## Features

- User registration and login
- Add and manage property listings
- Upload property images
- Browse all available properties
- Search properties by location
- Property detail pages
- User dashboard

## Installation

1. Clone the repository

```
git clone https://github.com/YOUR_GITHUB_USERNAME/rentease.git
```

2. Navigate to the project folder

```
cd rentease
```

3. Install dependencies

```
npm install
```

4. Create a `.env` file in the root directory and add your environment variables.

5. Start the server

```
node server.js
```

6. Open in browser

```
http://localhost:5000
```
